ModelArt

Obras creadas a partir de un modelo KNN

Inicialmente, para construir la colección de obras creadas a partir de diferentes técnicas de aprendizaje estadístico, se tomó como referencia e inspiración las obras generadas por el paquete de R aRtsy. Así entonces, la primera obra de esta colección fué creada a partir de un modelo KNN usando dos variables discretas en el rango 0-200. El modelo se ajustó tomando k = 200 vecinos más cercanos ya que en intentos anteriores se descubrió que, con un número pequeño elegido para k, no se obtenían aquellos "cúmulos" o aglomeraciones deseadas para efectos visuales. Finalmente, para dar un efecto de "nube" a través de una luz cálida, se escogen los colores dorado y blanco optando por interpolar linealmente la respuesta del modelo para así eliminar fronteras de cúmulos demasiado definidas.

La siguiente obra fué muy similar a la primera; usando el método de clasificación KNN, pero esta vez sin elegir el parámetro k (vecinos más cercanos para clasificar). De igual forma se creó un par de covariables aleatorias en el rango 0-200 y un vector de etiquetas (respuesta) con los colores a graficar para ajustar dicho modelo. Seguidamente se realizan las predicciones correspondientes usando datos generados de forma secuencial en el rango de las covariables originales, y se dispone el gráfico con la diferencia que en esta ocasión no se interpolan los límites de los conglomerados para obtener un efecto de "salpicadura". La paleta de colores elegida para esta obra recorre los colores café, blanco, beige y negro para obtener tonalidades "tierra" u "orgánicas".


Obras creadas a partir de modelos de ensamble

Seguidamente, la tercera obra de la colección fué generada a partir de un modelo RandomForest, donde en un procedimiento muy similar al anterior, se generaron dos variables aleatorias distribuídas Uniforme(0, 200) y una variable respuesta con las etiquetas "blanco" y "negro" que después facilitaría la implementación gráfica. Posteriormente se ajusta el modelo RandomForest sin parámetros tuneados, pues para este caso no es de interés dicho procedimento. Una vez obtenido el modelo, se producen dos nuevas covariables aleatoriamente en el rango original de los datos para con estas hacer la predicción respectiva. Finalmente, como se mencionó arriba, la variable respuesta fué creada con las etiquetas "blanco" y "negro", y con esta especificación se construyó el gráfico correspondiente obteniendo un toque "industrial monocromático"

Finalmente, la última obra creada usando métodos de aprendizaje estadístico, fué generada usando los datos predichos por un modelo de bagging con un comportamiento un tanto peculiar. La creación tanto de las covariables y la respuesta como de las predicciones es totalmente homóloga al procedimiento descrito anteriormente. Sin embargo, lo interesante de esta obra reside en el comportamiento del modelo hallado, pues este último aunque fué ajustado usando dos covariables "x" e "y", sólo toma en cuenta los valores de "x" para realizar las predicciones y sobre los valores de "y", el modelo se mantiene constante. De forma visual, este fenómeno se manifiesta con líneas que atraviezan el lienzo de forma completamente vertical, asemejándose de este modo a un código de barras. Los colores elegidos para este caso son intensos de colores cálidos para resaltar las características de la obra. El mal comportamiento del modelo conseguido usando bagging puede deberse a inconsistencias en la construcción del mismo y con esto a un pésimo ajuste de este a los datos originales, sin embargo para esta causa ese aspecto no es de interés y lo que centra la atención es el llamativo efecto visual conseguido.


AleatorizArte

En este apartado, se utilizan algunas distribuciones de probabilidad univariadas como ayuda para construir distribuciones normales multivariadas las cuales posteriormente son usadas para construir armonógrafos espectrales.
Un armonógrafo es una máquina mecanica que utiliza pendulos para crear imágenes geométricas [1]; por otro lado, un espectro armónico es un espectro que contiene únicamente componentes de frecuencia cuyas frecuencias son multiplos enteros de la frecuencia fundamental. [2]
Sea $\omega$ la frecuencia fundamental, entonces un espectro armónico es de la forma \[\{\cdots, -2\omega, \ \omega, \ 0, \ \omega, \ 2\omega, \cdots\}\] Si bien los armonógrafos espectrales se construyen con máquinas, estas pueden ser simuladas con softwares computacionales como python.

Simulación del proceso

Para la simular armonógrafos espectrales se utilizaron las librerias pygame, pygame.locals, sys, random, numpy, time, re y os, además de las funciones pi y sin de la librería math.
La librería pygame se utilizo para desplegar una ventana que simule el proceso de trazado de cada curva, el resto librerías así como sus respectivas funciones/métodos fueron usadas para la creación de las curvas. Las curvas fueron generadas con funciones seno en los ejes x e y, siendo representadas de la siguiente manera \[x = A_1Sin(\omega_1 t_1 + \varphi_1) + A_2Sin(\omega_2 t_2 + \varphi_2)\] \[y = A_3Sin(\omega_3 t_3 + \varphi_3) + A_4Sin(\omega_4 t_4 + \varphi_4)\]

Para la generación de $x$ e $y$ considere el vector aleatorio $\eta = \begin{bmatrix} \omega_1 \ \omega_2 \ \omega_3 \ \omega_4 \end{bmatrix}^T$ \[\eta \sim N\left(\begin{bmatrix} 0 \\ 0 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} \sigma_1^2 \ 0\ 0\ 0 \\ 0 \ \sigma_2^2 \ 0 \ 0 \\ 0 \ 0 \ \sigma_3^2 \ 0 \\ 0 \ 0 \ 0 \ \sigma_4^2 \end{bmatrix}\right)\] con $\sigma_i^2 \sim U(0, \ 1)$, $\varphi_i \sim U(0, 2\pi)$, $A_i \sim U_D[-4, 4]$ para $i = 1,2,3,4$
Finalmente, mediante un proceso iterativo se generó una secuencia de valores para $x$ e $y$ y se obtuvieron las siguientes imágenes.
